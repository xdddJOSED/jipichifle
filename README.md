# 🍌 Jipichifle - Landing Page

¡Bienvenido al repositorio de la landing page oficial de **Jipichifle**! 
Este proyecto es un sitio web estático moderno, diseñado para promocionar los tradicionales y crujientes chifles de Jipijapa.

## 🚀 Características Principales

* **Diseño Moderno y Responsive:** Interfaz atractiva tipo "Oreo", adaptable a dispositivos móviles y escritorio.
* **Sección "Desde la Fábrica":** Galería que muestra el proceso de elaboración de los chifles.
* **Chatbot Integrado:** Widget interactivo de atención al cliente en la esquina inferior derecha.

## 🛠️ Tecnologías Utilizadas

* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript vainilla.
* **Despliegue:** [Render](https://render.com/) (Static Site).
